# Sales Analysis – Power BI Dashboard

## Problem Statement
Analyze sales performance to identify revenue trends, high-performing categories, and regional insights that support business decision-making.

## Key KPIs
- Total Sales
- Total Profit
- Quantity Sold
- Profit Margin
- Sales Trend Over Time

## Dashboard Highlights
- Category-wise and region-wise sales analysis
- Time-series trends for sales and profit
- Contribution analysis of products and customers
- Interactive slicers for dynamic exploration

## Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Cleaning & Transformation

## Business Insights
- Identified top-performing categories contributing the majority of revenue
- Detected low-profit regions despite high sales volume
- Observed seasonal trends impacting monthly sales

## Files
- `sales.pbix` – Power BI report file

## How to Use
1. Download the PBIX file
2. Open using Power BI Desktop
3. Refresh data if required
